# SPGA (Sub-Par Golf Association) Fantasy Golf League

PGA Tour fantasy golf league dashboard for me and my friends. Pulls historical and active tournament data from the ESPN API. Currently only supports the majors and requires manual roster input.

**app.js** - ESPN API + Scoring Engine + UI Controller\
**data.js** - Manage Rosters + Managers + Points + Active Tournament\
**index.html / style.css** - HTML UI + CSS style sheet
